# ⌨️ Keybinds

Sway uses i3-compatible keybindings. The main modifier is `Mod4` (Super/Windows key).

## 🖥️ Programs & System

| Keybind | Action |
| :--- | :--- |
| `Mod + Enter` | Open Terminal (`lxterminal`) |
| `Mod + D` | Open App Launcher (Rofi) |
| `Mod + M` | Open Music Rofi Menu |
| `Mod + C` | Open Firefox Nightly |
| `Mod + T` | Open File Manager (`pcmanfm`) |
| `Mod + N` | Open Color Picker |
| `Mod + Q` | Kill focused window |
| `Print` | Fullscreen Screenshot |
| `Mod + Alt + R` | Reload Sway config |
| `Mod + Alt + E` | Power Menu (wlogout) |

## 🔍 Focus & Movement

| Keybind | Action |
| :--- | :--- |
| `Mod + H/J/K/L` | Focus Left/Down/Up/Right (Vim keys) |
| `Mod + Arrows` | Focus Left/Down/Up/Right |
| `Mod + Shift + H/J/K/L` | Move window Left/Down/Up/Right |
| `Mod + Shift + Arrows` | Move window Left/Down/Up/Right |

## 🗂️ Workspaces

| Keybind | Action |
| :--- | :--- |
| `Mod + [1-0]` | Switch to workspace 1-10 |
| `Mod + Shift + [1-0]` | Move window to workspace 1-10 |
| `Mod + Minus` | Show Scratchpad |
| `Mod + Shift + Minus` | Move window to Scratchpad |

## 📐 Layout & Resizing

| Keybind | Action |
| :--- | :--- |
| `Mod + B / V` | Split Horizontal / Vertical |
| `Mod + S` | Stacking layout |
| `Mod + W` | Tabbed layout |
| `Mod + E` | Toggle split layout |
| `Mod + F` | Toggle fullscreen |
| `Mod + Shift + Space` | Toggle floating |
| `Mod + Space` | Focus between tiling & floating |
| `Mod + R` | Enter Resize mode (Esc to exit) |

## 🔊 Media Keys

| Keybind | Action |
| :--- | :--- |
| `Volume Up/Down` | Volume 5% +/- (`pamixer`) |
| `Mute` | Toggle Master Mute |
| `Brightness Up/Down` | Backlight +/- (`brightnessctl`) |
| `Play/Pause/Next/Prev` | Media control (`playerctl`) |
